Weather Application

This project was built using HTML, CSS, JavaScript, API and Bootstrap. This application can search for weather anywhere in the world using OpenWeatherMap API, as well as display the future forecast and weather parameters of searched cities. Hosted on Netlify. Thanks to a succesful OpenWeatherMap API integration, the app displays real-time weather data of various cities in the world. You can easily change to your preffered location, by entering the city name in the search bar on the top. It is also possible to show the weather conditions of your current location , by simply clicking on the location button. Tools: InVision Freehand, HTML, CSS, JS, API, Responsive Web Design
